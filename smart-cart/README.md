# Smart Cart

Smart Cart is a simple web application designed to help users manage their shopping lists efficiently. This project includes a basic HTML, CSS, and JavaScript implementation for adding, removing, and viewing items in a shopping cart.

## Features
- Add items to your shopping cart
- Remove items from the cart
- View your current shopping list
- Responsive and clean user interface

## Project Structure
```
smart-cart/
  index.html      # Main HTML file
  script.js       # JavaScript logic
  style.css       # Stylesheet
```

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Start adding items to your shopping cart!

## Customization
- Modify `style.css` to change the appearance.
- Update `script.js` to add more features or change logic.
